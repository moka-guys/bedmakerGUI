Transcript/Gene Sense Check
What does this tool do?

This tool verifies that transcript IDs match their expected genes by:

  1. Taking a list of gene,transcript_id pairs
  2. Querying Ensembl's REST API to check which gene each transcript belongs to
  3. Comparing the expected gene with the actual gene

Useful for catching transcript ID errors before generating BED files.
